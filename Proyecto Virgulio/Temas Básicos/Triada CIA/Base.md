Portada:
	Los tres pilares base de la ciberseguridad son la Confidencialidad, Integridad y Disponibilidad
	Estos nos dan las bases para llevar a cabo un perímetro de seguridad robusto y funcional dentro de cualquier organización.
Slide 1:
	La Ciberseguridad tiene una variedad increíble de ramas, pero debemos recordar que la base de todas estas es la triada CIA (Por sus siglas en inglés).
	Estas pautan la manera en que se manejan las distintas amenazas que suelen surgir, como por ejemplo cómo asegurar una alta disponibilidad pese a sufrir ataques DoS, o cómo verificar que el dueño de ciertos datos es el único con acceso a esta información.
Slide 2:
	Pueden no parecer mucho pero en caso de faltar alguno, se puede caer todo.
	Piénsalo de esta manera, si tenemos una mesa de tres patas, si una falla o "le falta" perdemos todo y cae.
	Si protegemos un servidor con mil contraseñas, la confidencialidad estaría cubierta pero podríamos herir a la disponibilidad ya que el servidor podría alentarse con tantos procesos de validación.
