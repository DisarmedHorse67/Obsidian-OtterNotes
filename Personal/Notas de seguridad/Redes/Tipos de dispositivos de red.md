**Router**: Este comparte información basado en direcciones físicas (MAC) o en caso de redes TCP/IP se usan direcciones IP. Al navegar por internet, la petición llega al router, el DNS traduce el dominio a una dirección IP y la lanza a la web.

**Hub**: Este conecta dispositivos dentro de una misma red local, se les suele decir que son tontos ya que al recibir datos por un puerto lo retransmiten a todos los demás sin saber a quién se le mandó específicamente esos datos.

**Switch**:Este es una versión inteligente del hub, usa las direcciones MAC para saber a quién debe mandarle cada bit de información que recibe. Este obtiene las direcciones MAC mediante ARP.

**Bridge**: Este conecta redes separadas para hacerla una única red más grande, este trabaja en la capa 2 del modelo OSI.

**Firewall**: Este es un dispositivo que provee seguridad de red fundamental al monitorear el trafico entrante y saliente, mediante reglas previamente establecidas que nos permiten un control más granular de la red al crear redes privadas donde controlamos perfectamente lo que entra y sale de estas. Los firewalls pueden ser tanto software como dispositivos conectados a la infraestructura de red.
